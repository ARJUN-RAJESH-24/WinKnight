WinKnight v2.0.0 - System Guardian
===================================

Thank you for installing WinKnight!

WHAT'S NEW IN v2.0:
- Real-time CPU and GPU temperature monitoring
- 6 beautiful theme presets (Dark, Light, Ocean, Forest, Sunset, Nord)
- Network monitoring with speed tracking
- Quick actions (Flush DNS, Reset Network, Clear Temp)
- Power management features
- Improved performance and stability

REQUIREMENTS:
- Windows 10 or later
- .NET 8.0 Runtime (will be prompted if not installed)
- Administrator privileges for full functionality

FEATURES:
- System temperature monitoring
- RAM and disk usage tracking
- Startup program management
- System restore point creation
- SFC and DISM system repair
- BSOD history tracking
- Cache cleaning utilities

For support or issues, visit the project repository.

NOTE: Run WinKnight as Administrator for full functionality.
Hardware monitoring requires admin privileges to access sensor data.
